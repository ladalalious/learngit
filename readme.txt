嘿，今天我在学如何使用Git，这真是太有趣了！！
一个很迷人的分布式版本控制系统～～
不过让人遗憾的是在Windows上使用Git,并不能保存Word,Excel这些二进制文件的改动记录，不然所有工作文件及其改动信息都可以用Git进行文件管理，就好像我修改了一份原始记录或一份别的什么工作文件，别人也修改了这份原始记录或工作文件，到了未来某个时候，我觉得还是我第一次修改的那份文件好用，用Git的话就能非常方便地在修改过的不同的原始记录之间跳转，非常方便，可惜Git无法跟踪Word，Excel文件的改动～
